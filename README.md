# PCB Projects

A collection of custom electronic PCB designs created with KiCad.

## Projects

### 1. Basic DC DC Boost Converter
Simple open-loop boost converter using 555 timer and potentiometer to adjust output voltage.

### 2. DC DC Buck Converter
Closed-loop buck converter using LM2596 chip with potentiometer for voltage adjustment.

### 3. DC Motor Driver DIP
Bidirectional DC motor driver with H-bridge using Dual Inline Package (DIP) components.

### 4. DC Motor Driver SMD
Bidirectional DC motor driver with H-bridge using Surface Mount Device (SMD) components.

### 5. ESC BLDC Motor Driver
Electronic speed controller for brushless DC motors using Dual Inline Package DIP components.

### 6. PID Controller Using Opamps (DC Motor Position Control _ Potentiometer Sensor)
PID controller with operational amplifiers; Kp, Ki, Kd adjustable by potentiometers.

---

Each project folder contains the circuit schematic in PDF format, photos of the PCB layers, and 3D render images.
